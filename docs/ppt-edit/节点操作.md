# 节点操作

画布上的「节点」对应数据模型中的 `SlideElement`（`type`: `text` | `image` | `shape` | `table`）。选中节点后，可通过 Ribbon、浮动工具栏、右侧属性面板、右键菜单进行编辑；所有变更最终写入 `slide.elements[]`。

---

## 节点样式三层：Fill / Border / Font

参考 Google Slides，节点外观由三类样式组成，均保存在 `SlideElement.style`（`ElementStyle`）中：

| 层级 | 含义 | 常用字段 |
|------|------|----------|
| **Fill Color** | 节点填充 / 背景色 | 见下表「Fill 字段映射」 |
| **Border** | 边框颜色与线宽 | `borderColor`、`borderWidth` |
| **Font** | 字体、字号、颜色、段落 | `fontFamily`、`fontSize`、`fontWeight` 等 |

### Fill 字段映射（按类型）

| 节点类型 | Fill 字段 | 说明 |
|----------|-----------|------|
| 文本框 `text` | `style.background` | 文本框容器背景；未设置时使用半透明白底 |
| 形状 `shape` | `style.fill` | 面状形状填充；线型形状无 Fill |
| 图片 `image` | — | 无 Fill，仅 Border |
| 表格 `table` | 单元格 `style.background` | 在表格工具栏按单元格设置 |

### Border 字段（通用）

| 字段 | 类型 | 说明 |
|------|------|------|
| `borderColor` | string (HEX) | 边框颜色，6 位不含 `#` |
| `borderWidth` | number | 边框宽度（px）；设为 0 或清除颜色可隐藏边框 |

**操作提示**：双击 Fill / Border 颜色选择器可清除该配置，恢复默认。

### Font 字段（文本 / 表格）

| 字段 | 说明 |
|------|------|
| `fontFamily` | 字体族 |
| `fontSize` | 字号 8–96 |
| `fontWeight` | `normal` / `bold` |
| `fontStyle` | `normal` / `italic` |
| `underline` | 下划线 |
| `strikethrough` | 删除线 |
| `color` | 文字颜色 |
| `highlight` | 文本突出显示（高亮底色） |
| `align` | `left` / `center` / `right` |
| `bullets` | 项目符号列表 |

## 选中与交互

| 操作 | 说明 |
|------|------|
| 单击 | 选中单个节点，显示蓝色描边与 8 个缩放手柄 |
| Shift + 单击 | 多选 / 取消选中 |
| 双击文本框 | 进入文本编辑（`contentEditable`） |
| 拖动 | 移动位置；拖动手柄调整宽高 |
| 框选 | 空白处拖动框选多个节点 |
| 悬停 | 未选中时显示灰色描边 |
| Delete / Backspace | 删除选中节点（编辑文本时不触发） |

文本编辑中：Enter 换行，Esc 或点击画布外部保存并退出编辑。

## 样式配置入口

| 入口 | Fill / Border | Font |
|------|---------------|------|
| Ribbon **开始 → 文本** | 文本框 Fill + Border | 完整 Font 控件 |
| 画布浮动 **TextToolbar** | 同上 | 同上 |
| 画布浮动 **ShapeToolbar** | Fill + Border + 透明度 | — |
| 画布浮动 **TableToolbar** | 单元格背景 | 单元格 Font |
| 右侧 **属性** 面板 | 按类型展示 FillBorderControls | 文本 / 表格 |

## 按类型的样式配置

### 文本框（`text`）

**Fill & Border**

| UI | 数据字段 | 说明 |
|----|----------|------|
| 🪣 填充色 | `style.background` | 文本框 Fill Color |
| 边框色 | `style.borderColor` | Border 颜色 |
| 边框宽度 | `style.borderWidth` | Border 线宽 (px) |

**Font**

| UI | 数据字段 |
|----|----------|
| 字体下拉 | `fontFamily` |
| − / 字号 / + | `fontSize` |
| B / I / U / S | `fontWeight` / `fontStyle` / `underline` / `strikethrough` |
| A + 色条 | `color` |
| 🖍 + 色条 | `highlight` |
| 左 / 中 / 右 | `align` |
| • | `bullets` |

**内容**：双击画布编辑，或属性面板修改 `content`；高度随内容自适应。

> **内边距说明**：文本与边框之间保留 `10px 12px` 间距（在 `.freeform-text-display` 上），**Fill / Border 作用在外层容器**，填充色会铺满至边框内侧，不再出现「白边」。

---

### 形状（`shape`）

| UI | 数据字段 | 说明 |
|----|----------|------|
| 填充色 | `style.fill` | Fill Color |
| 边框 / 线条色 | `style.borderColor` | 线型形状为线条颜色 |
| 线宽 | `style.borderWidth` | px |
| 透明度 | `style.opacity` | 0.1–1 |
| 形状类型 | `style.shapeKind` | 矩形、椭圆、箭头等 |

---

### 图片（`image`）

| UI | 数据字段 | 说明 |
|----|----------|------|
| 边框色 | `style.borderColor` | Border |
| 边框宽度 | `style.borderWidth` | px |
| 图片源 | `imagePath` | URL 或 Base64 |

---

### 表格（`table`）

| 层级 | Fill | Border | Font |
|------|------|--------|------|
| 表格整体 | — | `style.borderColor` | `style.fontSize`、`style.align` |
| 单元格 | `cells[r][c].style.background` | — | `cells[r][c].style.*` |

## 数据示例

```json
{
  "id": "el-xxx",
  "type": "text",
  "x": 12,
  "y": 18,
  "w": 40,
  "h": 22,
  "content": "要点 1\n要点 2",
  "style": {
    "background": "FFFFFF",
    "borderColor": "CBD5E1",
    "borderWidth": 1,
    "fontSize": 16,
    "fontFamily": "\"Microsoft YaHei\", sans-serif",
    "fontWeight": "bold",
    "fontStyle": "normal",
    "underline": false,
    "strikethrough": false,
    "color": "344054",
    "highlight": "FFFF00",
    "align": "left",
    "bullets": true
  },
  "zIndex": 1,
  "rotation": 0,
  "locked": false
}
```

右侧 **JSON** 面板可查看完整 `SlideContent` / `PresentationContent` 数据。
